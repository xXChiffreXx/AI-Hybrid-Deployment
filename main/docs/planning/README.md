# Summer Project Planning Documentation

This folder translates the architecture, budget, and confidence documents into a concrete Summer 2026 pilot plan for the polypharmacy project.

## Documents

- [`00_summer_project_brief.md`](00_summer_project_brief.md): professor-facing summary of the recommended summer pilot profile, scope, and success criteria.
- [`01_quantization_strategy.md`](01_quantization_strategy.md): explicit quantization policy, allowed alternates, and promotion rules.
- [`02_pilot_track_a_scope.md`](02_pilot_track_a_scope.md): frozen Track A artifact scope for the summer pilot.
- [`03_mvp_backlog.md`](03_mvp_backlog.md): prioritized implementation backlog for turning the design repo into a runnable MVP.
- [`04_evaluation_protocol.md`](04_evaluation_protocol.md): operational review procedure derived from the confidence docs.
- [`05_telemetry_pilot_plan.md`](05_telemetry_pilot_plan.md): short pilot plan for replacing modeled assumptions with observed measurements.
